# پروژه گرافیت (Graphite)

## معرفی

این یک پلتفرم آگهی آنلاین مدرن است که با **Vue.js (Nuxt 3)** و **Tailwind CSS** ساخته شده است. تمرکز این پروژه بر ارائه یک تجربه کاربری بی‌نظیر، طراحی مینیمال (تم دارک) و عملکرد بالاست.

برای مشاهده نقشه راه دقیق و جزئیات فنی پروژه، لطفاً فایل `blueprint.md` را مطالعه کنید.

## راه‌اندازی پروژه

این پروژه برای اجرا در محیط Firebase Studio (IDX) پیکربندی شده است. مراحل راه‌اندازی به صورت خودکار انجام می‌شود:

1.  **نصب وابستگی‌ها:** دستور `npm install` به صورت خودکار هنگام باز شدن پروژه اجرا می‌شود.
2.  **اجرای سرور توسعه:** دستور `npm run dev` برای نمایش پیش‌نمایش زنده اجرا می‌شود.

## ساختار پروژه

- **`components/`**: کامپوننت‌های قابل استفاده مجدد Vue.
- **`layouts/`**: طرح‌بندی‌های کلی برنامه.
- **`pages/`**: صفحات اصلی و مسیرهای (routes) برنامه.
- **`assets/`**: فایل‌های استاتیک مانند CSS یا تصاویر.
- **`public/`**: فایل‌هایی که مستقیماً در ریشه سرور قرار می‌گیرند.
